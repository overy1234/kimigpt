📋 개인정보처리방침

최종 수정일: 2025년 10월 14일

1. 수집하는 정보

1.1 자동 수집 정보

• IP 주소
• 브라우저 정보
• 접속 시간 및 방문 기록
• 쿠키 (선택사항)

1.2 사용자 제공 정보

• 채팅 대화 내용 (임시 저장, 세션 종료 시 삭제)

2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:
• 서비스 제공 및 개선
• 사용량 분석 및 통계
• 서비스 남용 방지
• 법적 요구사항 준수

3. 정보 보관 및 삭제

• 대화 내용: 브라우저 세션 종료 시 즉시 삭제
• 로그 데이터: 7일 보관 후 자동 삭제
• 쿠키: 사용자가 직접 삭제 가능

4. 제3자 제공

수집된 정보는 다음을 제외하고 제3자에게 제공되지 않습니다:
• Hugging Face (AI 모델 API 제공업체)
• Vercel (호스팅 서비스 제공업체)
• 법적 요구가 있는 경우

5. 사용자 권리

사용자는 다음 권리를 가집니다:
• 정보 열람 요청
• 정보 삭제 요청
• 서비스 이용 거부

6. 보안

우리는 사용자 정보 보호를 위해 다음 조치를 취합니다:
• HTTPS 암호화 통신
• API 키 보안 관리
• 정기적인 보안 점검

7. 쿠키 정책

본 서비스는 다음 목적으로 쿠키를 사용할 수 있습니다:
• 세션 유지
• 사용자 경험 개선
• 광고 (Google AdSense)
